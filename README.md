# Pokédex made with Vue.js
Show information about a pokem
[View in browser](https://sharp-euler-148c78.netlify.app/)

# What i used for this project
- [Vue.js (The Progressive JavaScript Framework)](https://vuejs.org/)
- [PokéAPI (The RESTful Pokémon API)](https://pokeapi.co/)
- [axios (Promise based HTTP client for the browser and node.js)](https://github.com/axios/axios/)
- [Color Thief (Grab the color palette from an image using just Javascript)](https://github.com/lokesh/color-thief/)
